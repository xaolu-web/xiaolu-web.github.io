HTML 简介

HTML（超文本标记语言）是用于创建网页的标准标记语言。它通过一系列的标签来描述网页的结构和内容，浏览器会读取这些标签并将其渲染成可视化的网页。例如，<html> 标签是整个网页的根标签，所有其他标签都嵌套在它内部。

基础标签

标题标签

HTML 提供了六个级别的标题标签，从 <h1> 到 <h6>，<h1> 通常用于表示最重要的标题，字号最大，而 <h6> 字号最小，用于表示相对次要的标题。例如：

<h1>这是一级标题</h1> <h2>这是二级标题</h2>

段落标签

使用 <p> 标签来定义段落，段落之间会自动换行，并且有一定的上下间距。例如：

<p>这是一段文字，它会在网页上独立成段显示。</p>

链接标签

通过 <a> 标签可以创建超链接，让用户能够从一个页面跳转到另一个页面或资源。例如：

<a href="https://www.example.com">点击跳转到示例网站</a>，其中 href 属性指定了链接的目标地址。

图片标签

<img> 标签用于在网页中插入图片，需要通过 src 属性指定图片的路径，以及 alt 属性提供图片的替代文本，当图片无法加载时，替代文本会显示出来。例如：

<img src="your_image.jpg" alt="这是一张示例图片">

列表标签

无序列表

<ul> 标签用于创建无序列表，列表项使用 <li> 标签包裹。例如：

<ul> <li>列表项 1</li> <li>列表项 2</li> <li>列表项 3</li> </ul>

有序列表

<ol> 标签用于创建有序列表，同样列表项使用 <li> 标签包裹，列表项会自动按顺序编号。例如：

<ol> <li>第一项</li> <li>第二项</li> <li>第三项</li> </ol>

表格标签

<table> 标签用于创建表格，表格由 <tr>（表格行）、<td>（表格单元格）和 <th>（表头单元格）等标签组成。例如：

<table> <tr> <th>表头 1</th> <th>表头 2</th> </tr> <tr> <td>数据 1</td> <td>数据 2</td> </tr> </table>
